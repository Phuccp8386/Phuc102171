# Thư mục Audio

Thư mục này chứa các file audio về Chủ tịch Hồ Chí Minh.

## Cấu trúc thư mục:
- `songs/` - Các bài hát về Bác Hồ
- `speeches/` - Các bài phát biểu của Bác

## Định dạng hỗ trợ:
- MP3 (.mp3)
- OGG (.ogg)
- WAV (.wav)

## Cách thêm file audio:

1. Đặt file audio vào thư mục phù hợp
2. Cập nhật đường dẫn trong HTML:
   ```html
   <source src="assets/audio/songs/ten-file.mp3" type="audio/mpeg">
   ```

## File mẫu cần có:
- `NhuCoBacHo.mp3` - Bài hát "Như Có Bác Hồ Trong Ngày Vui Đại Thắng"
- `BacHoLongChung.mp3` - Bài hát "Bác Hồ Lòng Chung"
- `TienQuanCa.mp3` - Bài hát "Tiến Quân Ca"

## Lưu ý:
- File audio nên có chất lượng tốt nhưng kích thước hợp lý
- Nên có cả định dạng MP3 và OGG để tương thích tốt với các trình duyệt
